In my JavaScript part of project I tried solving a quadratic equation to find its roots. I use three input boxes to collect the user input of three coefficients of a quadratic equation. Then I used a button which when clicked calls a function that takes the values from the input fields and pass them as parameters to other function which calculated roots for the equation. I used java script document object model and button event handling for the whole process.

In this process I found if difficult to handle different outputs with different datatypes and I tries JSON stringify and JSON parse to convert objects to strings ans vice-versa
